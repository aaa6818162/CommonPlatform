退换货流程

1、符合退换货条件，可以致电我们全国服务热线4008-816-616，我们的售后服务人员在了解并核实情况后，会确认退换货； 非商品质量原因引起的退换货，我们的售后服务人员会查实二次包装、 运输、安装等费用和您确认，安排退换货； 是商品质量原因引起的退换货，我们的售后服务人员会查实质量问题，如确实属于质量问题，直接确认退换货。

2、我们处理退换货的正常内部流程工作时间约为2周。实际情况根据退换货的不同情况以及您所在的不同地域位置而定，两者均会和您协商处理。

3、退换货确认手续操作完毕后，财务安排退款事宜的正常工作时间为1周。根据退款支付方式和您所在的区域的不同，您收到退款的时间将会有所不同。

退换货政策
"滨雅商城的退换货服务

您所购买的商品，退换货有效期为收货后10天内，以原始收货凭证上的收货日期为准在商品保质期内，因商品本身质量原因的退换货（包括客户收货时未发现的质量瑕疵），依据《中华人民共和国消费者权益保护法》三包规定承担相应的退货、换货或修理的义务。

1.以下任一情况在退换货有效期内，可办理免费退换货手续：

①正常状态商品在订单配送前；

②正常状态商品在订单配送后，且同时符合以下条件：

a.商品含有完好包装、配件及原包装内的说明书、三包卡、合格证齐全，同时无污渍褪色等情况，不影响二次销售；

b.除贴身使用的床上用品（床垫、枕头、被芯、枕芯、被套、枕套等）以及卫浴产品私人产品（毛巾、浴巾等）；

c.未产生运费、退换货服务费用等。

2.以下任一情况在退换货有效期内，确认商品不影响二次销售的情况下，可办理有偿退换货手续；

①商品配送后办理退换货，需要我们提供退货服务，门店所在城市免费配送范围内，需收取商品订单金额5%的退换货服务费，若超出门店所在城市免费配送范围，需收取商品订单金额5%的退换货服务费及实际产生的运费。

②定制状态商品办理退换货，包括但不限于沙发类特定面料定制，家具类定制，窗帘类定制或特殊情况下为您安排生产的产品等，需要收取定制商品订单金额50%作为违约金。

3.以下任一情况拒绝退换货：

①您不能出示任何一项退换货凭证（销售单、销售小票、配送单）；

<②您选购的商品超过了退换货有效期，且商品无质量问题；

③处理降价的产品包括不限于陈列样、有瑕疵或残损等商品，门店设计顾问会明示降价或瑕疵的原因并书面说明，您对我们书面的瑕疵为由的退货；

④.贴身使用的床上用品（床垫、枕头、被芯、枕芯、被套、枕套等）以及卫浴产品私人用品（毛巾、浴巾等）

符合退换货条件后，可按以下流程办理退换货手续

1.大件商品（包括家具、沙发、大型灯具、地毯等），请致电全国服务热线，售后服务人员将上门鉴定，在确认符合退换货条件后，在3个工作日内将商品退回；或顾客自行选择第三方物流运回购买地，您可持有效的原始退换货凭证至原购物门店办理对应的退换货手续；

2.小件商品（包括纺织品、饰品、台灯等），在保持产品和包装完好并可二次销售的情况下，您可携带商品及有效的原始退换货凭证至原购物门店，设计顾问核实产品符合退换货标准后即可办理相应的退换货手续。"

2、公司汇款或支票支付的订单，需与客服人员确认公司相关信息后进行处理, 我们均会原路退回至客户原支付的公司账户中，在收到退换货后3-5个工作日处理，给您带来的不便请您谅解。

3、由于第三方银联或银行处理的时间或相关原因，造成退款未及时到账，请您与我们的客服联系，我们会立刻帮助解决。退换货地址

售后保养服务
售后养护服务范围

家具保养

沙发清洗

服务对象

持白金卡、钻石卡会员

白金卡会员：免费养护服务券一张（有效期一年）

钻石卡会员：免费养护服务券两张（有效期一年）

适合地区：门店所在城市

服务券发放流程

1、 累计消费达15万元（含）以上的客户，您可以致电客服中心（400-888-1916）申请养护服务券事宜，客服中心和您确认后将寄送服务券；

2、 保养产品以会员登记地址为准，沙发、家具同时保养按1次计次（限门店所在城市服务范围内）。

3、 保养时间最好在产品送到使用后的8~10个月为最佳，梅雨季节不建议做保养。

服务内容

1、 木制类、金属类、皮类、藤类家具：

上门检查，维修、补漆、上蜡，螺丝导轨加固，表面清洁，门铰链导轨等活动五金配件上油。

2、 地毯类：

上门检查，清洁。小面积勾丝可以进行修复，具体操作按专业技师上门后判定。

3、 沙发类：

上门检查，维修、补漆，上蜡，螺丝加固，表面清洁，五金配件上油。

温馨提示：

1 ) 未出示本公司工作牌的，请拒绝服务，谨防假冒；

2 ) 与本公司工作人员私下交易出现问题的，本公司将不负任何责任；

3 ) 清洁机器工作时会有噪音产生，请您谅解；

4 ) 沙发、地毯类表面污渍建议最好在3天内联系上门进行维护保养，避免形成顽固性污渍，已经形成顽固性污渍的部位，不能保证完全清洁；

5 ) 清洁服务后请您出示会员服务卡，工作人员将进行回收。

由于时间问题，我们的服务可能存在不完善的地方，恳请您提出宝贵意见和建议。服务过程中出现任何问题请及时与我们的客服中心联系：4008-816-616，我们将竭诚为您服务！

服务券使用说明

1、 会员凭此券在购买一年内通过电话预约，由滨雅商城专业技师为您提供一次免费上门养护服务。

2、 养护产品类型为家具与地毯。

3、 养护内容：家具类检查、维护，沙发、地毯类清洁服务。

4、 维护产品以会员登记地址为准，沙发、家具、地毯同时维护的，按1次计次。

5、 保养时间在产品送到使用后的8~10个月为最佳。

6、 您电话预约后，我们会在安排好服务人员后，再和您沟通确定具体上门服务时间。
